# Challenge Bot MVP

Быстрый Telegram MVP для проверки гипотезы по боту микро-впечатлений.

## Что внутри
- `bot.py` — основной код бота
- `requirements.txt`
- `.env.example`
- `data/starter_challenges_50.json` — стартовые 50 челленджей
- `data/messages_ru.json` — тексты экранов
- `data/challenge_schema.json` — схема челленджей
- `data/user_state.json` — простое файловое хранение состояния

## Что умеет MVP
- `/start`
- быстрое меню
- выбор челленджа
- текущий челлендж
- завершение челленджа
- история эмоций
- оценка эмоции
- кастомный челлендж
- share with friend через deep link

## Быстрый запуск
1. Создай бота через BotFather и получи токен.
2. Создай виртуальное окружение:
   - `python -m venv .venv`
   - `source .venv/bin/activate` (Mac/Linux)
   - `.venv\Scripts\activate` (Windows)
3. Установи зависимости:
   - `pip install -r requirements.txt`
4. Скопируй `.env.example` в `.env` и подставь токен.
5. Запусти:
   - `python bot.py`

## Что менять руками быстрее всего
- Контент челленджей: `data/starter_challenges_50.json`
- Тексты экранов: `data/messages_ru.json`

## Как использовать Google Sheet как простую CMS позже
Самый быстрый путь:
- хранить мастер-таблицу челленджей в Google Sheet
- раз в день или вручную экспортировать CSV/JSON
- заменять `starter_challenges_50.json`
- перезапускать бота

## Какие метрики смотреть

- переслал другу

## Ограничения MVP
- данные пользователей пока в JSON-файле
- нет нормальной БД
- нет настоящей админки
- нет аналитики в отдельной панели
- нет quiet hours и продвинутой модерации
